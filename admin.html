💡 How to publish a new post
  1. Fill in the Write Post tab completely
  2. Click Save Post to Database — this stores it in Supabase
  3. Come to this tab and click Generate Code
  4. Copy the code and paste it into your index.html posts array
  5. Re-deploy to Vercel — done! 🎉
